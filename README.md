AMAZING FOOD.


 This website is designed to be responsive and accessible on a range of devices, making it easy to navigate for potential food enthusiasts that would like to contribute, access and share interesting recipe ideas.

# DESIGN
Colour Scheme
The two main colours on Amazing Food are  orange and white.The advantage of using orange color in food is its strong link to beneficial nutrients, particularly carotenoids like beta-carotene, which is converted to vitamin A. This color is associated with various health benefits, including improved vision and skin health, a boosted immune system
